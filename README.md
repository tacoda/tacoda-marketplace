# tacoda-marketplace

Claude Code plugin marketplace by [tacoda](https://github.com/tacoda).

## Add the marketplace

```
/plugin marketplace add tacoda/tacoda-marketplace
```

## Plugins

| Plugin | Description |
|---|---|
| [bridle](https://github.com/tacoda/bridle) | Agent project harness manager for Claude Code |

## Install a plugin

```
/plugin install <plugin-name>@tacoda-marketplace
```

For example:

```
/plugin install bridle@tacoda-marketplace
```
